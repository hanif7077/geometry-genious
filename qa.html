JS questions and answer

1.What is DOM and what is the purpose of DOM?

The abbreviated of Document Object Model is known as DOM. which is a programming interface for web documents.It represents the page so that programs can change the document structure, style, and content.

2.How will you select HTML elements using DOM? Name the DOM methods.

I have to Select HTML DOM elements by the name attribute, using the javascript getElementsByName(), getElementById() and getElementsByClassName() method.

3.What is event bubble?

The bubbling is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.

4.What is a callback function and why will you use it?

A callback's purpose is to execute code in response to an event. These events can be like mouse clicks; with callback, we can add text at the end of each statement like execute this code every time the user clicks a key on the keyboard.
